TrainerX

A modpack of cheat mods, including:
Новые читы - Автор: Archer30
Чит-мод - Автор: Valery
Отображение событий - Авторы: Valery, Gamecreator
Выполнение ERM-команды - Автор: Hawaiing
Выберите поле боя - Автор: Valery, Archer30

Горячие клавиши для Карты Приключений:
Нажмите F2, чтобы изменить свойства любого героя/игрока.
Нажмите F3, чтобы разрешить/запретить удаление объектов и событий, затем перемещайте Героев.
	щелкните правой кнопкой мыши любой объект, чтобы открыть меню сканера объектов.
	Щелкните, удерживая Alt, чтобы повторить последнее действие.
	Alt + num, чтобы быстро изменить параметры.
Нажмите F6, чтобы использовать дополнительные читы.
Нажмите X, чтобы выполнить ERM-команды.

Горячая клавиша для battlefiled:
Нажмите F2, чтобы изменить тип специальных ландшафтов.

TrainerX можно загрузить во время игры. Нажмите F12, чтобы обновить настройки мода.
(Если у вас возникли проблемы с некоторыми описаниями после нажатия F12, сохраните и загрузите снова)

Совместимость: Advanced Classes Mod (вторичные навыки уровня M / GM), Prima (первичные навыки до 249) и Third Upgrade Mod (новые существа и артефакты).
