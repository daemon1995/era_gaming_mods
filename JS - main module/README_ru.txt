Мод представляет собой набор шаблонов, плагинов, опций, фиксов, доработок, регулируемых через json:
/Lang/jsmainmodule_settings.json


Для корректной работы мод требует:
- Era Erm Framework
- Game Enhancement Mod
- Wog Fix Lite


Список новых шаблонов для случайных карт:
- M TestTempl (для тестирования)
- XL Slater Ring
- XXL Big Ring (требуется мод XXL)


Список плагинов:
- Accumulate_Creatures_In_Dwellings.dll - если внешнее жилище захвачено, то оно накапливает существ от недели к неделе
- always_3_army_stack_and_2_lvl_creature_building_built.dll - игрок всегда стартует с полной армией, а стартовый замок игрока - с отстроенным строением существ 2 уровня
- ConfluxDwellingType20.era - внешнее жилище существ сопряжения с 4 слотами дает прирост +1 в сопряжениях игрока всем четырем элементалям, а не только воздушным
- HotA_Pathfinding_and_WayfarersBoots.era - Поиск Пути работает как в HotA; добавлен артефакт из HotA - Сапоги Странника (требуется саб-модуль JS - emerald)
- HotA_SpecBonus.era - специализации героев по существам, заклинаниям работают как в HotA
- InstantAnim.dll - добавляет постоянную анимацию существам в бою, как в HotA
- RMG_MapGenerationFixes.era - устанавливает ширину всех проходов между зонами на случайных картах в одну клетку


Список новой графики для объектов:
- AVSutop5.def - Утопия Драконов для болотистой почвы
- AVSutop8.def - Утопия Драконов для лавовой почвы
- AVXfyth7.def - Фонтан Молодости для подземной почвы
- AVTmystd.def - Мистический Сад для грязевой почвы
- avtprosp.def - Мистический Сад для песчаной и подземной почвы
- AVXstbl7.def - Конюшни для лавовой и подземной почвы
- AVXtrek2.def - Древо Мудрости для песчаной почвы
- AVXtrek5.def - Древо Мудрости для болотистой почвы
- AVXtrek3.def - Древо Мудрости для каменистой почвы (взято из базовой сборки WoG)
- AVXtrek7.def - Древо Мудрости для лавовой и подземной почвы
- HAExit01.def - Врата Подземного Мира для подземной почвы
- avmoilpm.def - Водяное Колесо для подземной почвы


Список опций:
- см. jsmainmodule_settings.json: список содержит более 30 опций, каждая опция имеет описание действия и совместимости с WoG-опциями
- чтобы выключить опцию, нужно поменять значение ключа "value" с true на false, и наоборот


Настраиваемая вогификация артефактов и знамен:
- см. jsmainmodule_wogify.json: упорядочена вогификация артефактов и знамен, реализована кастомизация процесса: можно указать, какие типы артефактов заменять и с каким шансом


Остальные изменения:
- изменены AI_Value монстров согласно HotA
- изменены value, density и ограничение на зону объектов согласно HotA
- добавлена способность контролируемого выстрела по площади существам с флагом MON_FLAG_SPLASH_SHOOTER при нажатии клавиши G
- запрещена некромантия из элементалей и механизмов
- запрещены "водные" артефакты, заклинания и Навигация на "сухопутных" картах, герои со специализацией Навигации заменяются на HotA-героев (Беатрис, Кинкерия)
- Песочные Часы Недоброго Часа нейтрализуют удачу только если она положительная
- добавлены звуки для способностей частичного и полного блока
- добавлена анимация для полного блока
- анимация заклинания минного поля ускорена
- анимация заклинания взрыва заменена на HotA
