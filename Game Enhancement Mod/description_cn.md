**游戏增强模组**（简称 GEM）

作者：daemon_n

本模组基于插件和脚本，引入了诸多改动：

视觉效果改动：

- 每个英雄职业在冒险地图和战斗中拥有独特的外观；

- 每个城市等级在冒险地图中拥有独特的外观；

- 战斗中显示单位生命条（可选）；

- 扩展生物信息对话框；

- 扩展战斗中小队信息预览对话框；

- 显示大型资源数值；

- 在所有城市预览中显示城市建筑状态；

- 当屏幕宽度足够（860）时显示秘银图标；

- 缩小“下一个英雄”按钮的宽度；

- 添加“设置菜单”按钮，用于打开模组设置菜单（快捷键“~”）；

- 在小地图附近添加一个面板，用于显示和管理快速战斗设置（可选）；

- 在显示神器描述提示信息时显示主要属性加成（可选）；

- 在显示神器描述提示时，显示组合神器部件（可选）；

- 在城镇窗口中打开附加设置菜单的按钮；

游戏玩法变更：

- 在城镇窗口中删除英雄；

- 法术研究（可选）；

- 按下 ALT 键时，在冒险地图对象上方显示其名称提示（可选）；

游戏交互快捷键：

- F1：打开帮助菜单；

- ALT（在冒险地图上）：在对象上方显示提示；

- ALT（在冒险地图上）：在状态栏显示路径长度；

- ALT+鼠标右键（在冒险地图上）：在单独的消息中显示提示；

- ~（在冒险地图上）：打开模组选项菜单；

- ~（在城镇窗口中）：打开附加城镇按钮菜单；

其他引入永久性变更的插件和补丁：

- accumulate_8lv_creatures.bin - 允许您在其住所中累积 8 级生物；

- accumulate_santagremlins_creatures.bin - 允许你在圣诞老人小精灵的住所中积累它们；

- ArtGuard.dll - 防止你直接在怪物旁边偷取神器和卷轴；

- BattleAIvsAIFix.dll - 允许AI在战斗中获得正确的经验值和骷髅数量；

- BattleReplay.dll - 允许你重玩战斗！太棒了！！！；

- BattleSave.dll - 在战斗前创建一个名为“战斗！”的存档文件；

- draw soils.bin - 正确显示土壤、道路和景观物体；

- fast upgrade stacks in town.dll - 使用“A”键+鼠标左键快速升级生物；

- H3.RMGDescription.dll - 正确显示随机地图名称和描述；

- StartArmyAllSlots.bin - 所有英雄初始都有 3 个生物栏位；
