**Game Enchacement Mod** (aka GEM)
Author: daemon_n

I present a mod that introduces numerous changes based on plugins and scripts:

Visual Changes:
- Each hero class has a unique appearance on the Adventure Map and in battle;
- Each city level has a unique appearance on the Adventure Map;
- Unit health bars in battle (optional);
- Expanded creature information dialog;
- Expanded squad information preview dialog in battle;
- Display of huge resource values;
- Display of city building status in all city previews;
- Display of the mithril icon when the screen is wide enough (860)
- Reduced the width of the "Next Hero" button;
- Added a "Settings menu" button that opens the mod settings menu (hotkey "~");
- Added a panel with buttons for displaying and managing quick battle settings near the minimap (optional);
- Display primary stat bonuses when displaying tooltips with artifact descriptions (optional);
- Display combo artifact parts when displaying tooltips with artifact descriptions (optional);
- Button to open the additional settings menu in the town window;

Gameplay changes:
- Delete a hero in the town window;
- Spell research (optional);
- Display tooltips with the names of Adventure Map objects directly above them when pressing the ALT key (optional);

Hotkeys for interacting with the game:
- F1: Open the help menu;
- ALT (on the Adventure Map): Display tooltips above objects;
- ALT (on the Adventure Map): Display path length in the status bar;
- ALT+RMB (on the Adventure Map): Display a tooltip in a separate message;
- ~ (on the Adventure Map): opens the mod options menu;
- ~ (in the town window): opens the menu of additional town buttons;

Other plugins and patches that introduce permanent changes:
- accumulate_8lv_creatures.bin - allows you to accumulate level 8 creatures in their dwellings;
- accumulate_santagremlins_creatures.bin - allows you to accumulate Santa Gremlins in their dwellings;
- ArtGuard.dll - prevents you from stealing artifacts and scrolls directly next to monsters;
- BattleAIvsAIFix.dll - allows the AI ​​to receive the correct amount of experience and skeletons in battles;
- BattleReplay.dll - allows you to replay battles! HOORAY!!!;
- BattleSave.dll - creates a save file named ``BATTLE!`` before a battle;
- draw soils.bin - correct display of soil, roads, and landscape objects;
- fast upgrade stacks in town.dll - quick creature upgrades with "A" + LMB;
- H3.RMGDescription.dll - correct display of random map names and descriptions;
- StartArmyAllSlots.bin - all heroes start with 3 creature slots;
